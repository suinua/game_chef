# Game Chef
PvPゲーム作成を補助するプラグインです  
ANNIやCorePVP,TeamDeatMatch,ワンパン弓、FFA,etc.. 
こういったPVPベースのゲームを簡単に作れます  
[特徴・機能(未完成)](https://github.com/suinua/game_chef/tree/master/doc/Index.md)
